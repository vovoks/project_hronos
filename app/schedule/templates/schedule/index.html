Добро пожаловать в проект Хронос!

Хронос - предоставляет REST-API интерфейс для внешних сервисов в целью упрощения работы с расписанием МИЭТа.

REST-API сервис расписания

  1. Для начала работы Вам потребуется зарегистрировать собственный сервис по ссылке. После регистрации Вам придет сообщение на электронную почту с реквизитами для подключения к сервису.
  2. Для проверки реквизитов и получения информации о составе медотов API можно перейти по ссылке.
    • Чтобы выполнять запросы к сервису потребуется получить access_token. Для этого, через метод /token передайте (кнопка "Try it out"), полученные username и password и нажимте Execute. В ответ Вам должен вернуться access_token.
    • Скопируйте значение access_token и нажмите кнопку Authorize и в поле value введите Bearer "скопированный access_token" (без кавычек), далее нажмите кнопку Authorize. Теперь все запросы на этой странице будут выполняться с указанным access_token.
    • Выполните метод /schedule, чтобы убедиться, что сервис работает. В ответ в случае успеха должно вернуться расписание.
    • Важно! Access_token имеет срок действия (60 минут). Поэтому, каждые 60 мин его требуется обновлять с помощью refresh_token, который выдается при получении acсess_token. Чтобы обновить access_token требуется вызвать метод /token/refresh/ и передать ему в качестве тела запроса refresh_token. Подробнее с методом аутентификации JWT Вы можеет ознакомиться по ссылке.
